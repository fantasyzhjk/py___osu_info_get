### 如何获取相关信息🚀️ 
 
玩家id:  
打开自己的主页，网址最后的数字就是  
比如https://osu.ppy.sh/users/4504101  
4504101就是id  
.    
oauth：   
在自己个人资料的设置界面，下拉到最后  
创建新的oauth应用程序  
回调链接随意  
.  
铺面bid:  
知道铺面的网址，如https://osu.ppy.sh/beatmapsets/1250654#osu/2675038  
2675038就是bid  
如果是https://osu.ppy.sh/beatmapsets/1250654  
那么可以先复制到浏览器打开。  
1250654不是bid，那是sid
